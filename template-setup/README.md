# GreenCross.App Solution Template

## What is this?

This is a production-ready .NET 10 solution template featuring clean architecture, designed for Windows Forms applications.

## Quick Start

After using this template to create your repository:

```powershell
.\template-setup\rename-template.ps1 -NewName "YourCompany.YourApp"
```

## Features Included

- Clean Architecture (Onion Architecture)
- Entity Framework Core with SQL Server
- Repository Pattern with Unit of Work
- Dependency Injection
- High-performance logging
- Configuration management
